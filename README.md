# Exploratory maze

A maze game created with the Pyxel game engine.

*Note: This game was created for me personally to study Pyxel. As such, the code is very messy and unoptimized.*

## Requirement

- Python 3.8.10
- [Pyxel](https://github.com/kitao/pyxel) 1.4.3
